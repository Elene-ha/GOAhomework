რომელ ორ ნაწილს მოიცავს Web Development-ი?

Web Development მოიცავს ორ ძირითად ნაწილს:

Frontend Development

ვებსაიტის ის ნაწილი, რასაც მომხმარებელი ხედავს და ურთიერთობს.

Backend Development

ვებსაიტის ფარული ნაწილი, რომელიც მუშაობს სერვერზე და მართავს მონაცემებს, ფუნქციონალს და მონაცემთა ბაზებთან კავშირს.

Frontend Development-ის კომპონენტები და მათი დანიშნულება:

HTML (HyperText Markup Language)

ქმნის ვებსაიტის სტრუქტურას.
დანიშნულება: განსაზღვრავს ელემენტებს, როგორიცაა სათაურები, პარაგრაფები, სიები და ბმულები.

CSS (Cascading Style Sheets)

განსაზღვრავს ვებსაიტის ვიზუალურ დიზაინს.
დანიშნულება: აძლევს ფერს, ზომას, განლაგებასა და შრიფტებს ვებსაიტზე.

JavaScript

აძლევს ვებსაიტს ფუნქციონალსა და ინტერაქტიულობას.
დანიშნულება: ამატებს მოქმედებებს, მაგალითად, ღილაკზე დაჭერისას რაიმე ცვლილება.
